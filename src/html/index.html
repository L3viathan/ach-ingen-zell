-ești, -ov, -oara

A visual exploration of the spatial patterns in the endings of Romanian town and village names.

This is a fork of Moritz Stefaner's -ach, -ingen, -zell. The suffixes were chosen more or less subjectively, with consultation by @sarnthil. The data is the same geonames dataset.

Darker colors indicate a higher proportion of place names with the respective suffix in a geographic cell. Hover over a cell for a while to see a tooltip with the place names.

Please note: My approach is not 100% scientific, as I only match the letters at the end of the string, not actual syllables.

Find the code at: http://github.com/L3viathan/esti-ov-oara
